# california-housing-dataset
The California Housing Dataset is a popular dataset used in machine learning to predict house prices in California. It was originally based on data from the 1990 U.S. Census.

Objective

The main goal is to build a regression model to predict the median house value in a given district based on various features such as income, location, and housing characteristics.

This dataset is commonly used for:
Regression modeling
Feature engineering
Data visualization
Machine learning practice

Typical Workflow:
Data Cleaning (handle missing values)
Exploratory Data Analysis (EDA)
Feature Engineering
Model Training (Linear Regression, Random Forest, etc.)
Model Evaluation
